‚ö° Quick Start

  1. Install dependencies 
    npm install
    Make sure Hardhat and ethers are installed for contract deployment.
  2. Compile the contract 
    npx hardhat compile
  3. Deploy the contract 
    npx hardhat run scripts/deploySafeWallet.ts --network YOUR_NETWORK
    Note: The contract deploys with owner 0xFDf84a0e7D07bC56f7De56696fc409704cC83a24 as the signer. Replace YOUR_NETWORK with mainnet, polygon, etc. as needed.
  4. Frontend Usage 
    import { getSafeSigner } from "./utils/signer";
    The frontend will automatically use 0xFDf84a0e7D07bC56f7De56696fc409704cC83a24 as the wallet for all contract interactions.
